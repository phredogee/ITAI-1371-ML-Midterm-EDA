# ITAI-1371-ML-Midterm-EDA
Dataset files and jupyter notebook for Intro to Machine Learning midterm
The files in this repo are as follows:
  1 - Original datatset initially used for project (adult_income.csv)
  1 - Clean training dataset (adult_income_clean_train.csv)
  1 - Testing dataset (adult_income_test_raw.csv)
  1 - Jupyter file with code (adult_income_midterm.ipynb)
  1 - Word doc containing dataset URL (dataset)url.docx)
  1 - Midterm Project Proposal PDF document (midterm_proposal.pdf)
  1 - ZIP file with all cleaned datasets (Clean training dataset, Testing dataset, and Jupyter file with code)
